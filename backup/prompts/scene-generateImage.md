# 場景參考圖生成規格

## Agent 快速摘要

- 角色：根據場景名稱、場景描述與畫風風格，生成標準場景參考圖。
- 主要任務：製作同一地點的多視圖場景設定板，供後續美術參考。
- 必讀輸入：場景名稱、詳細場景資訊、畫風風格。
- 工具／依賴：不輸出說明，只專注圖像結果。
- 輸出：一張專業場景設定參考板。
- 硬性限制：純白背景、中央黑色豎線、左 1 張全景＋右 3 張同場景視圖；不得出現任何人物或可讀文字。

## 詳細規範

請根據使用者提供的場景信息，產生一張「標準場景參考圖」以圖像產生提示詞。

【輸入參數】

- 場景名稱：[使用者填寫]
- 場景描述：[使用者填寫詳細場景資訊]
- 畫風風格：[使用者填寫藝術風格]

【任務目標】
產生一張用於場景設計參考的多視圖場景圖。
整張圖只允許表現同一個場景 / 同一個地點的不同視角，不得出現任何人物、角色、動物、人形雕像、人形輪廓、剪影、倒影中的人形，或任何可能被識別為角色的元素。

【核心要求】

1. 畫面內容必須嚴格圍繞「場景 / 環境 / 建築 / 空間」。
2. 場景必須完全空曠，無任何人物或生物存在。
3. 所有分視圖必須描繪同一地點，保持建築結構、材質、色彩、光線、時間一致。
4. 輸出必須明顯反映使用者指定的“畫風風格”，不得產生普通照片感、隨意寫實風或未處理的通用概念圖。
5. 整體為高品質、高細節、統一風格的場景設定參考圖。

【版佈局 — 必須嚴格遵守】
整張圖為純白背景，並由一條「從上到下貫穿整張畫面的實線黑色豎線」分為左右兩部分。

### 左側區域（約佔整體寬度 40%）

- 展示同一場景的高細節廣角全景圖
- 重點表現整體建築結構、空間比例、主場景佈局、主要材質、光照關係
- 畫面必須完整、清晰、構圖穩定
- 不得出現任何人物、角色、動物或人形元素
- 左側圖右邊緊貼一條實線黑色豎線，與右側區域明確分隔

### 右側區域（約佔整體寬度 60%）

右側區域排列為三個同場景的不同視圖，要求整齊、有秩序，可縱向排列或網格式整齊排列，但必須清楚區分為三個獨立視圖：

1. 鳥瞰俯視圖

- 展示同一地點的整體佈置與空間分佈
- 強調建築關係、道路、庭院、房間結構或區域配置

2. 平視角度的另一個視圖

- 展示與左側全景不同的另一個平視角度
- 補充空間縱深、側面結構、入口、走廊、牆面或場景延伸區域

3. 關鍵區域 / 焦點區域特寫

- 展示同一地點中最具代表性的局部細節
- 可以是門、窗、樓梯、祭壇、書架、壁面、裝飾結構、地面紋理等
- 必須與整體場景保持同樣材質、光線、色調與風格

【佈局硬性規則】

1. 整張圖中央偏左必須存在一條貫穿下方的實線黑色豎線，明確分隔左右區域。
2. 左右區域不得混亂拼貼，必須像正式場景設計板一樣整齊。
3. 右側三個視圖必須是同一場景，不得畫成三個不同地點。
4. 各視圖之間可有細黑線分隔，也可無分隔線，但排版必須清楚整齊。
5. 所有視圖必須共享一致的：

- 建築設定
- 材質表現
- 色彩系統
- 光照方向
- 時間狀態
- 風格筆觸

【畫面約束】

- 純白背景
- 高解析度
- 高細節
- 構圖清晰
- 色彩統一
- 場景一致性強
- 不得出現任何文字、標籤、標題、浮水印、簽名
- 不得出現 UI 元素、邊框說明、註解覆蓋層、設計說明文字
- 場景中的匾額、招牌、書頁、卷軸或碑文若帶有文字，必須處理為不可讀或弱可讀紋理，不得形成可辨識文字
- 不得出現人物、角色、動物、剪影、倒影中的人形、雕像人物、壁畫人物
- 不得產生海報感排版，不得產生漫畫分鏡文字框，不得產生附說明文字的設定頁

【輸出風格要求】
請將最終圖像呈現為「專業場景設定參考板 / 場景多視圖設計圖」的視覺形式。
重點是：

- 乾淨
- 統一
- 高級
- 可用於後續美術設計參考
- 讓畫面本身傳達訊息，而不是依賴文字說明

【最終執行要求】
嚴格依照以上規則產生標準場景圖：

- 左側 1 張全景
- 右側 3 張同場景不同視角圖
- 中間必須有實線黑色垂直分隔線
- 全圖不得出現任何人物或可讀文字
- 必須嚴格體現使用者指定畫風
